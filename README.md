# Horror Movie Themed Quizzes

Minimalist black-paper-white React quiz game with an old-school video game feel.

## Features

- Pick one movie quiz or choose **All Movies**.
- 20-question round every run.
- 4 choices per question, only 1 correct answer.
- Correct answer gives a ding sound.
- Wrong answer gives an ehh sound and reveals the right answer.
- Timer ticks while waiting for an answer.
- Scoreboard tracks correct answers, wrong answers, streak, accuracy, and difficulty.
- Wrong limit ends the game.
- Correct streaks increase difficulty.
- Missed questions are saved in localStorage and appear more often later.

## Movie banks

- Chucky
- Halloween
- Scream
- Midsommar
- Five Nights at Freddy's

